## 💻 Sobre o Projeto

Este projeto utiliza o poder do **Microsoft Copilot** para criar uma jornada de mentoria automatizada. Através de prompts estruturados, transformamos a IA em dois especialistas que trabalham em conjunto para planejar sua transição ou evolução de carreira técnica.

## 🛠️ O que deve ser feito (Fluxo de Trabalho)

O sucesso do projeto depende da execução sequencial dos dois agentes integrados:

### 1. Fase de Diagnóstico (AGENT 1)
- **Ação:** Copie o conteúdo do arquivo `AGENT 1 - Entrevistador de Carreira em Tecnologia` e cole no chat do Copilot.
- **O que acontece:** O Copilot assumirá a postura de um recrutador técnico e iniciará uma entrevista de 7 perguntas. 
- **Atenção:** Responda às perguntas uma por uma para que a IA processe seu perfil adequadamente. Ao final, ela sugerirá as 3 melhores carreiras para você.

### 2. Fase de Planejamento (AGENT 2)
- **Ação:** Após escolher uma das carreiras sugeridas, copie o conteúdo do arquivo `AGENT 2 - Planejador de Carreiras` e cole no Copilot.
- **O que acontece:** O Copilot mudará de papel para um Mentor de Estudos.
- **Resultado:** Ele entregará um cronograma detalhado de 90 dias, incluindo o que estudar semanalmente, sugestões de projetos para portfólio e links para cursos na plataforma DIO.

## 🎯 Resultados Esperados

Ao seguir os passos acima no Copilot, você terá:
- Um diagnóstico preciso das suas afinidades profissionais.
- Um roadmap de 3 meses adaptado à sua disponibilidade de tempo.
- Uma lista de competências técnicas (Hard Skills) e comportamentais (Soft Skills) prioritárias.

## 📚 Pré-requisitos

- Acesso ao **Microsoft Copilot** (via navegador ou Windows).
- Os arquivos de prompt deste repositório.
- Responder às etapas da entrevista.
